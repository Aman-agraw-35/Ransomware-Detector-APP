# SeqDefender Frontend (React + Vite)

## Run

1. Install dependencies:

```powershell
cd frontend
npm install
```

2. Start dev server:

```powershell
npm run dev
```

The UI will call http://localhost:8000 by default.

To change API base URL, set:

- VITE_API_BASE in frontend/.env

Example:

```text
VITE_API_BASE=http://localhost:8000
```
